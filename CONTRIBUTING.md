## CSS 선택자 규칙

1. Layout ( display, Position... )
2. Box ( Margin, Padding, Border-box... )
3. Background ( Background... )
4. Font (Font-family, Font-Size... )
5. Animation (keyframe, transition, transform... ) 
6. ETC

+파일명은 케밥케이스로 합니다.
+변수, 함수는 캐멀케이스로 작성합니다.
+들여쓰기는 4칸 띄어서 적습니다.
